## Technical Details

### Image Optimization Strategy
- **Format Priority**: WebP > AVIF > JPEG/PNG (with fallbacks)
- **Max Widths**: 
  - Hero: 1920px
  - Gallery: 800px
  - Thumbnails: 400px
- **Quality**: 80-85% for photos
- **Lazy Loading**: All images below the fold

### Instagram Integration Options

#### Option A: Instagram Basic Display API
**Pros**: Free, full control, real-time updates
**Cons**: Requires app setup, token management, rate limits
**Setup Time**: 30-40 minutes

#### Option B: Elfsight Widget
**Pros**: No code, auto-updates, easy setup
**Cons**: Paid service ($6-12/month), less customization
**Setup Time**: 10-15 minutes

#### Option C: Manual/Static
**Pros**: Fastest setup, no API dependencies
**Cons**: Manual updates required, no engagement metrics
**Setup Time**: 15-20 minutes
